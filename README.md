# 🖥️ Mini-HTOP em C++

Um monitor de sistema estilo **htop**, escrito em **C++** com **ncurses**.  

Mostra:
- Uso de CPU (total + por núcleo)
- Uso de memória
- Uso de disco
- Top 5 processos consumidores de CPU

---

## 🚀 Como rodar

### Instalar dependências
```bash
sudo apt update
sudo apt install g++ libncurses5-dev libncursesw5-dev -y
